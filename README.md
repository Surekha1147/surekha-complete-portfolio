# 🌐 Personal Portfolio Website

This repository contains my **personal portfolio website**, developed as part of the **Thiranex Virtual Internship (Ref: THX-MAY1426-898)**.  
It showcases my education, skills, projects, internships, certifications, coding profiles, and achievements in a modern, accessible, and fully responsive design.

---

## ✨ Features
- **Semantic HTML5 structure** (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`)
- **WCAG accessibility compliance** with ARIA labels and roles
- **SEO‑friendly meta tags** (title, description, keywords, OpenGraph)
- **Responsive architecture**:
  - CSS Grid for complex layouts
  - Flexbox for component alignment
  - Mobile‑first media queries
- **Advanced styling**:
  - Animated gradient backgrounds
  - Glassmorphism/neumorphism effects for cards
  - Responsive typography scaling with `clamp()`
  - Light/Dark mode toggle using CSS variables
  - Smooth hover transitions and micro‑interactions
  - Scroll‑based animations (fade‑in, slide‑up)
- **Hero section redesign**:
  - Personal photo in circular frame with hover zoom
  - Name, tagline, and summary
  - Call‑to‑action button with gradient styling
  - Sticky navigation bar with blur/shadow
  - Scroll indicator for smooth navigation
- **Accessible contact form** with tab navigation and screen reader support
